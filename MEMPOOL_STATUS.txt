### Validation Pipeline

```
submit_tx(tx)
    ▼
Structure Check (tx.id != empty)
    ▼
UTXO Validation (not coinbase)
    ├─ Exists
    ├─ Sufficient
    └─ No double-spend
    ▼
Signature Validation
    └─ Non-empty structure
    ▼
Conflict Detection
    └─ No mempool conflicts
    ▼
Output Validation
    └─ All values > 0
    ▼
Added to mempool if valid
```

### Selection Algorithm

```
get_ready_txs()
    ▼ (filter: parents_satisfied)
Sort by fee_rate (BinaryHeap)
    ▼ (highest first, determined by tx_id if tied)
Topological Order
    ▼ (parents before children)
Enforce Block Size
    ▼ (stop when full)
Return ordered transactions
```
